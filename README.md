# EcoUz

**EcoUz** — ekologiya bilan bog‘liq murojaatlarni qabul qiluvchi va ularning ijrosini ta’minlash uchun ishlaydigan veb-loyiha. Foydalanuvchilar murojaatlarini yuboradi, admin va ijrochilar esa ularni ko‘rib chiqadi va natijalarni kuzatadi.  

## Asosiy funksiyalar

- Foydalanuvchilardan ekologiya bilan bog‘liq murojaatlarni qabul qilish.  
- Murojaatlarni ijrochilarga yo‘naltirish va monitoring qilish.  
- Natijalarni va tegishli rasm materiallarini saqlash va ko‘rsatish.  
- Foydalanuvchi va admin panel orqali murojaatlarni boshqarish.  

## Texnologiyalar

- Backend: **Python**, **Django**, **Django REST Framework**  
- Frontend: **HTML**, **CSS**, **JavaScript**  
- Ma’lumotlar bazasi: **PostgreSQL**  
- Versiya nazorati: **Git**
